# Visio Divina 專案開發規範

> [!IMPORTANT]
> **版本控管與推送規範**
> 1. **自動提交 (Local Commit)**：每次修改程式碼後，必須自動執行 `git add` 與 `git commit`。
> 2. **禁止自動推送 (No Auto-Push)**：除非使用者明確下達指令（例如：「Push 到 GitHub」、「發布到 Netlify」），否則禁止執行 `git push`。
> 3. **全域規則例外**：本專案規則優先於全域規則中的「自動 Push」設定。

## 其他注意事項
- 地圖標籤 (Map Labels) 應保持純文字高對比顯示，無背景方框。
- 專注模式標題應動態顯示書卷名稱。
